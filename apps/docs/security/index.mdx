## Security Controls Summary

| Control Area | Implementation | Status |
|-------------|----------------|--------|
| Authentication | Bcrypt (cost 12), MFA, Passkeys, SSO | ✅ Active |
| Authorization | RBAC with organization-level permissions | ✅ Active |
| Encryption | AES-256-GCM at rest, TLS 1.3 in transit | ✅ Active |
| Rate Limiting | 3-tier rate limiting with Arcjet WAF | ✅ Active |
| Input Validation | Zod schemas + DOMPurify | ✅ Active |
| Audit Logging | Structured logs with PII redaction | ✅ Active |
| Security Headers | CSP, HSTS, X-Frame-Options via Helmet | ✅ Active |
| Secret Management | Environment variables with validation | ✅ Active |

## Reporting Security Issues

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT** open a public GitHub issue
2. Email security concerns to the security team
3. Include detailed reproduction steps
4. Allow 90 days for remediation before public disclosure

## Security Updates

Security patches are released as soon as possible after discovery. Enable notifications for security advisories in the repository.
